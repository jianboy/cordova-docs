Tizen 平臺指南

本指南介紹了如何設置您的 SDK 開發環境部署科爾多瓦運行 Tizen 作業系統的設備的應用程式。

要求和支援

Tizen SDK 需要 Linux Ubuntu 10.04/10.10/11.04/11.10 （32 位） 或 Windows XP SP3/7 （32 位）。

開發人員應使用 cordova 實用程式 Tizen SDK 的結合。 命令列介面資訊，請參閱如何安裝它，添加專案，然後生成和部署專案。

安裝 SDK

從tizen.org下載 Tizen SDK.

在 SDK 中打開的專案

  1. 推出 Tizen Eclipse IDE。

  2. 選擇檔 → 導入 → Tizen Web 專案：

  3. 按下一步.

  4. 請確保選中了選擇根目錄。

  5. 請確保選中了複製到工作區中的專案。

  6. 按流覽並選擇科爾多瓦 Tizen samples 專案目錄 （如 /cordova-basic ）：

  7. 按完成。您的專案現在應該將其導入並出現在專案資源管理器視圖中：

部署到模擬程式

用滑鼠右鍵按一下專案資源管理器視圖中的專案並選擇運行作為 → Tizen Web 模擬器應用程式：

將部署到設備

  • 請確保目標設備正常啟動、 連接和配置。必須正確設置其日期和時間設置。

  • 使用連接資源管理器視圖選擇的應用程式部署目標：視窗顯示視圖 → 連接資源管理器.
