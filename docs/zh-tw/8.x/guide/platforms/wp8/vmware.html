配置 VMWare 融合

這一節演示如何配置 VMWare 融合在 Mac 上，以便您可以使用科爾多瓦生成 Windows Phone 應用程式。

Microsoft 開發人員網路提供有關如何運行 Windows 下 VMWare 融合的一般說明。 之後安裝 Windows，請執行以下步驟：

  1. 在 VMWare 融合內, 選擇您已編寫並選擇設置的 Windows 8 磁片映射.

  2. 選擇的處理器和記憶體的配置選項。 請確保指定兩個處理器內核，並啟用在此虛擬機器的虛擬機器管理程式應用程式：

    Windows Phone 模擬器單獨使用半位元組的記憶體，所以，總體來說，你應該為 VMWare 預留至少 2 GB。

  3. 選擇高級設置。啟用首選虛擬化引擎： 英特爾 VT-x 與 EPT選項：

  4. 修改.vmx檔，添加或修改以下設置：

    hypervisor.cpuid.v0 ="FALSE"mce.enable ="TRUE"vhv.enable ="TRUE"

一旦你完成這些步驟，你準備然後安裝 Windows Phone SDK。請參閱 Windows Phone 8 平臺指南的詳細資訊。
