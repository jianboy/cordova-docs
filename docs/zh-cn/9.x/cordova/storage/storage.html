存储

Cordova的存储选项的概述。

几个存储 Api 是对Cordova应用程式可用。请参阅 html5rocks。有关更完整的概述和示例。

认为

此 API 也已知作为web 存储、简单的存储，或由其备用会话存储介面，提供同步键/值对存储，是可用基础 web 视图实现中。 请参阅W3C 规范的详细资讯。

WebSQL

此 API 是在底层 web 视图中可用。 Web SQL 资料库规范提供了更多的功能全面的资料库表可以通过 SQL 查询访问。

以下平台支持 WebSQL：

  • Android 系统
  • 黑莓 10
  • iOS
  • Tizen

IndexedDB

此 API 是在底层 web 视图中可用。索引 DB提供更多的功能比 LocalStorage 但较少比 WebSQL。

以下平台支持 IndexedDB：

  • 黑莓 10
  • 火狐浏览器的作业系统
  • Windows Phone 8
  • Windows 8

基于外挂程式的选项

除了上面列出的 Api 的存储，档 API允许您对本地档案系统上的缓存资料。 其他的Cordova外挂程式提供类似的存储选项。
