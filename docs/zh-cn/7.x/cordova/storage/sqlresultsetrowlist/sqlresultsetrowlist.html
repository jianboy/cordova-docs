SQLResultSetRowList

属性之一的 [SQLResultSet](../sqlresultset/sqlresultset.html) 从 SQL 查询中包含的行返回。

属性

  • 长度： SQL 查询所返回的行数。

方法

  • 专案： 返回 JavaScript 物件所表示的指定索引处的行。

详细资讯

SQLResultSetRowList包含从 SQL 返回的资料 select 语句。 该物件包含 length 属性，该值多少行 select 语句返回。 若要获取的资料行，调用 item 方法，以指定一个索引。 它返回 JavaScript Object 其属性是的资料库列 select 反对执行语句。

支援的平台

  • Android 系统
  • 黑莓手机 WebWorks （OS 6.0 和更高）
  • iOS
  • Tizen
