快速入门

1

安装Cordova

Cordova的命令行运行在 Node.js 上面并且可以通过 NPM. 安装。 根据 平台具体指导 安装相应平台的依赖。打开命令提示符或终端，然后键入 npm install -g cordova.

$ npm install -g cordova
npm install -g cordova
2

创建一个项目

使用命令行创建一个空的Cordova项目。导航到你希望创建项目的目录，然后键入 cordova create <path>.

要知道这个命令完整的选项，键入 cordova help create.

$ cordova create MyApp
cordova create MyApp
3

添加一个平台

在创建完一个Cordova项目后，导航到项目目录。在项目目录中，你需要添加你想构建app的平台

为了添加平台， 键入 cordova platform add <platform name>.

为了知道你可以添加的平台，你可以运行 cordova platform.

$ cd MyApp

$ cordova platform add browser

cd MyApp cordova platform add browser
4

运行App

在命令中,键入 cordova run <platform name>.

$ cordova run browser
cordova run browser
5

Cordova包装你的HTML/JavaScript app到原生app容器中，可以让你访问每个平台设备的功能。这些功能通过统一的JavaScript API提供，让你轻松的编写一组代码运行在几乎市面上的所有手机和平板上，并可以发布到相应的app商城中。

Cordova工具

多样化的生态，包括命令行工具、JavaScript框架和云服务用来增强Cordova的功能

Adobe PhoneGap

PhoneGap是Apache Cordova最原始和最流行的分发。只需要几分钟，就可以让你的HTML,CSS和JavaScript变成你设备上的一个App，你只要简单的使用我们 桌面 和 开发者应用.

Ionic Framework

Ionic是一个前端SDK用来构建跨平台移动App。以Angular做基础，也提供平台集成服务，如消息推送和分析。

Monaca

综合云动力和框架无关的工具集。Monaca同时支持在线和离线的开发、动态加载调试和云构建体验。

Onsen UI

基于自定义元素的HTML5框架，提供大量可供选择的组件和响应式布局支持。Onsen UI可以让你设计专业的多平台应用，无需更多的技能。

Visual Studio

流行的IDE，用来为Android，iOS和Windows构建跨平台的App。凭借先进的构建和调试支持来完成。

App Builder

App Builder是适用于Microsoft Windows的完整IDE 无需编程知识即可创建HTML5和本机应用程序。商情 准备在您的应用中使用的数十个控件和操作 很多应用样本来学习它。

Framework7

Framework7是一个免费的开源移动HTML框架 使用iOS和Android开发混合移动应用程序或Web应用程序 原生的外观和感觉。

Evothings Studio

Evothings Studio为Cordova开发人员提供了快速发展 物联网的工作流程，教程和示例应用程序。 支持蓝牙低功耗（BLE）和其他物联网相关 技术。

NSB/AppStudio

NSB / AppStudio是一个用于webapps /本机应用程序的IDE。 一步安装 包括完整的PhoneGap集成，以及Bootstrap，jQuery Mobile 和jqWidgets。 拖放设计器。 在JavaScript中轻松编程 或BASIC。 Windows和MacOS。

Mobiscroll

Mobiscroll是跨平台UI的集合 提供优质iOS，Android和Windows Phone应用程序的控件。 框架不可知，使用普通的Javascript， jQuery，Angular，React或Knockout。

Instabug

Instabug为Cordova开发人员提供错误报告和应用程序内容 反馈解决方案 有一分钟的安装指南， 它使用户能够在自动附加时无缝地报告错误 网络日志，重复步骤等详细信息
